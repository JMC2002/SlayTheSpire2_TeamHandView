[h1]View Teammates' Hands in Multiplayer[/h1]
[h2] 1. Overview [/h2]
Having trouble communicating what cards you have in multiplayer? Want to guide your teammates on how to play their cards? Use this MOD and put some pressure on your teammates!

[url=https://github.com/JMC2002/SlayTheSpire2_TeamHandView] GitHub Repository [/url]

[h2] 2. Features [/h2]
[list]
    [*] In multiplayer, hover over another player's status bar to view their current hand. You can also use a configurable hotkey (set controller bindings in Steam Input) to lock the display
    [img]https://images.steamusercontent.com/ugc/12663845091139171176/D8E176940CBB9C3A0F377F33C8C86BD9B1C2625F/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
    [*] Supports adjusting card scale, maximum cards per row, and the lock/unlock hotkey in MOD settings
    [img]https://images.steamusercontent.com/ugc/17108105702485605775/F373F29E2C12C3BC918B901E3A3EC6956FDA4C12/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false[/img]
[/list]

[h2] 3. Notes [/h2]
[list]
    [*] Because of the game's mechanics, installing any MOD for the first time can cause save desynchronization. You can use [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3747533700] BetterSaveSlots [/url] to solve this
    [*] This MOD requires the following dependency:  
• [url=https://steamcommunity.com/sharedfiles/filedetails/?id=3747526103] JmcModLib [/url]  
    [*] The localization content of this MOD (including the workshop page and changelog), except Chinese, is generated by ChatGPT. It currently supports all languages officially supported by the game. If you think any wording is inaccurate or would like to contribute your own translation, feel free to open an issue / PR on GitHub or contact me directly
[/list]

[h2] 4. Compatibility [/h2]
[list]
    [*] This mod makes extensive use of reflection and may stop working after game updates. If you find that the mod no longer works, please report it promptly.
[/list]
