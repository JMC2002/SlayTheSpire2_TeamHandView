# Changelog

所有对本项目的重要更改都将记录在此文件中。

版本号规则：主版本号.次版本号.修订号，其中修订号每次修改后递增。

## [1.0.4] - 2026-4-30
### Added
新增可配置组合快捷键，用于锁定或解除锁定当前悬停显示的其他玩家手牌预览。

## [1.0.3] - 2026-4-30
### Fixed
改为引用 JmcModLib.Runtime.dll，并修正本地化文件的实际资源位置。

## [1.0.2] - 2026-4-30
### Fixed
修正 JML 配置本地化文件的资源目录。

## [1.0.1] - 2026-4-30
### Added
联机时悬停其他玩家状态栏显示其当前手牌。

## [1.0.0] - 2026-4-30
### Added
初始版本。
